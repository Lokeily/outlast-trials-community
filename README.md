# 逃生试炼复生方案（社区共建版）

《逃生：试炼》(The Outlast Trials) 强化剂（Amps）配装规划页，**由社区共同维护**。

- 三大类各 3×3 强化剂九宫格（工具 / 技能 / 药物），红框圈出推荐项
- 👍 = 社区推荐项
- 底部 💡 社区小贴士
- 页面底部 **Giscus 评论区**（基于 GitHub Discussions），可评论、分享配装

## 本地预览
直接用浏览器打开 `index.html` 即可（图标已 base64 内嵌，离线可用）。

## 如何参与共建
发现错误、想补充强化剂效果、或小贴士？欢迎提交 Pull Request：
1. Fork 本仓库
2. 修改 `index.html`
3. 发起 PR，维护者审核合并

图标来自官方 Wiki（outlast.fandom.com），社区建议整理自贴吧讨论。
